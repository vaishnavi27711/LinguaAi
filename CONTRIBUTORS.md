## Round 2 Build Plan — April 4–5, 2026 · COEP Campus

### Day 1 — Core Pipeline
- Document upload and parsing — PDF + DOCX with full structure preservation
- Source quality validation engine — spell check, consistency analysis, punctuation, formatting
- RAG pipeline — ChromaDB vector store, segment embeddings, exact/fuzzy/new classification
- LLM translation with glossary and style enforcement (Claude / Gemini / Ollama)

### Day 2 — UI + QA + Learning + Polish
- Side-by-side proofreading and multi-level approval editor with full audit trail
- Post-Translation QA — tag consistency, number accuracy, length validation
- Back-translation verification — three-way comparison view
- Continuous learning loop — TM update + glossary enrichment on every approval
- Format-preserved export (DOCX / PDF with original layout)
- Analytics dashboard — TM leverage rate, productivity, cost savings
- Final presentation and live demo before judges
